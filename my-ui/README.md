# hjl-btn
# install
`npm i hjl-btn --save`
# use
``
// main.js
import StVueUi from 'hjl-btn'
Vue.use(StVueUi)
``
# Attributes
|参数|说明|类型|可选值|默认值|
|:---|:--:|:--:|---:|---:|
|size|尺寸|string|medium/small/mini|—|
|type|类型|string|success/info|—|
|round|是否圆角按钮|boolean|—|false|
|disabled|是否禁用状态|boolean|—|false|
